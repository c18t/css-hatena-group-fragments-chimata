# css-hatena-group-fragments-chimnata
'巷の断片' のブログテーマ

## How to Develop
1. `yarn dev` します
2. Stylusに `@import url('http://127.0.0.1:8080/theme-fragments-chimata.css');` でスタイルを作ります
3. はてなを見ながらデザインします

## Project Making
```sh
$ gitignore code node windows macos
$ yarn init
$ yarn add stylus
$ yarn add superstatic
```
